# Grade 7 STEM Projects

> Project-Based Learning | Technology Education | Hands-On STEM Activities

---

# Overview

This repository showcases a collection of project-based learning activities that I designed and delivered while teaching Grade 7 Technology during the 2025–2026 academic year.

Each project was created to introduce students to engineering concepts through practical experimentation, encouraging creativity, critical thinking, teamwork, and problem-solving. Students actively assembled electronic circuits, programmed Arduino boards, tested hardware, and observed how software interacts with physical systems.

The objective was to transform the classroom into an engineering laboratory where students learn by designing, building, testing, and improving real projects.

---

# Teaching Philosophy

I believe students learn engineering best by building real systems.

Rather than relying only on theoretical explanations, every lesson was designed around hands-on activities that allow students to explore electronics, programming, and engineering design through experimentation and collaboration.

Students were encouraged to make mistakes, analyze results, troubleshoot problems, and improve their solutions—developing both technical knowledge and engineering thinking.

---

# Learning Objectives

These projects were designed to help students:

- Discover the fundamentals of electronics.
- Understand basic programming concepts.
- Develop logical and computational thinking.
- Learn engineering design through experimentation.
- Improve teamwork and communication.
- Build confidence using technology.
- Connect classroom theory with practical applications.

# Projects Included

## Blink

Students discovered the fundamentals of Arduino programming by controlling LEDs, understanding digital outputs, and writing their first embedded program.

### Skills Developed

- Arduino basics
- Digital outputs
- Programming logic
- Hardware connections

## Alarm System

Students designed an electronic alarm system using motion detection and a buzzer.

### Components

- Arduino
- PIR Motion Sensor
- Buzzer
- Breadboard
- Jumper Wires

### Skills Developed

- Sensor integration
- Digital inputs
- Embedded programming
- Troubleshooting

## Smart Barrier

Students built an automatic barrier controlled by an ultrasonic sensor and a servo motor.

### Components

- Arduino
- Ultrasonic Sensor
- Servo Motor
- Breadboard

### Skills Developed

- Distance measurement
- Automation concepts
- Servo control
- Engineering design

## Arduino Fundamentals

Students learned how to:

- Upload programs to Arduino boards.
- Read and understand simple code.
- Connect electronic components.

- Test and debug hardware.

## Multimeter Workshop

Students were introduced to the safe use of a digital multimeter.

Topics included:

- Measuring voltage
- Measuring resistance
- Measuring continuity
- Component identification
- Electrical safety

---

# Classroom Methodology

Each lesson followed the same engineering workflow:

1. Introduction to the engineering concept.
2. Demonstration by the teacher.
3. Circuit assembly by students.
4. Programming the Arduino board.
5. Hardware testing.
6. Troubleshooting.
7. Discussion and reflection.

Students actively participated in every stage of the project, from wiring circuits to uploading code and validating the final results.
